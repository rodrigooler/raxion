# Q4 2026 — Testnet Launch Plan (v0.1)

## Goal

Move from Q3 code-prep scaffolding to public Testnet operation with continuous reliability tracking.

## Scope

- Deploy rollup/token/poiq programs to public testnet environment
- Publish endpoints + explorer
- Execute external-agent onboarding wave
- Run first 30 days of the 90-day stability clock
- Submit first external security audit package

## Workstreams

1. Infrastructure
   - production-grade RPC, indexers, observability, backup, incident response
2. Protocol Ops
   - challenge/slash monitoring, dissent queue operations, proof-failure handling
3. Ecosystem
   - agent onboarding playbook and external pilot cohort
4. Security
   - audit remediation loop and pre-mainnet hardening

## Milestones

- M1: Public Testnet bootstrap complete
- M2: First external agents live
- M3: Audit 1 submitted and triage started
- M4: 30-day stability checkpoint passed

## Deliverables

- Public status dashboard and weekly reliability report
- Incident postmortem template and escalation matrix
- Mainnet readiness delta report (what remains after Q4)
